## 技术栈

- **前端**: Next.js 16, React 19, TypeScript
- **UI**: Ant Design, @lobehub/ui, antd-style
- **状态管理**: Zustand, SWR
- **数据库**: PostgreSQL, PGLite, Drizzle ORM
- **测试**: Vitest, Testing Library
- **包管理**: pnpm (monorepo)

## 贡献流程

1. Fork 仓库
2. 创建功能分支
3. 提交代码
4. 创建 Pull Request
5. 等待审核和合并

详细流程请查看 [开始开发](start.md)。
